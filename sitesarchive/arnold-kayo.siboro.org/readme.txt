This is the first public version of my guestbook.

Features:
	- only a tab-separated textfile is needed
	- multi-language support
	- support for css stylesheets
	- admin interface for deleting entries
	- email notification
	- the owner can write one comment to every entry
	
	
Getting started:
	- edit the variables in gbook.php
	- chmod 666 gb
	
Bugs: Please email me if you find any bugs.

Supported languages: Currently only English and German are supported, but it is very simple to add new languages.
